# 🛍️ Pérola Calçados - Projeto de Banco de Dados

## 📖 Sobre o Projeto

O projeto **Pérola Calçados** foi desenvolvido para a disciplina de Banco de Dados com o objetivo de modelar e implementar um sistema para uma loja virtual de calçados.

O sistema permite o gerenciamento de clientes, produtos, categorias, fornecedores, pedidos, pagamentos e entregas, garantindo a organização das informações e o controle das operações da loja.

---

## 🎯 Objetivo

Desenvolver um banco de dados relacional capaz de:

* Cadastrar clientes;
* Gerenciar produtos e categorias;
* Controlar fornecedores;
* Registrar pedidos;
* Gerenciar pagamentos;
* Acompanhar entregas;
* Consultar informações através de comandos SQL.

---

## 🗂️ Entidades do Projeto

* Cliente
* Categoria
* Produto
* Fornecedor
* Pedido
* Item_Pedido
* Pagamento
* Entrega

---

## 🔗 Relacionamentos

* Um cliente pode realizar vários pedidos.
* Um pedido pode possuir vários itens.
* Um produto pode aparecer em vários itens de pedido.
* Uma categoria possui vários produtos.
* Um fornecedor fornece vários produtos.
* Cada pedido possui um pagamento.
* Cada pedido possui uma entrega.

## 📌 Estrutura do Banco

O banco de dados foi modelado seguindo os princípios de normalização e utilizando chaves primárias (PK) e chaves estrangeiras (FK) para garantir a integridade dos dados.
